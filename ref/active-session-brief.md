  - dotted 점은 아주 미세하게 유지한다. 배경 surface는 흰색(`#fff`)에 아주 미세한 dot만 얹고, floating card는 투톤감이 약하게 살아나도록 `#f7f7f6` 단일 surface로 둔다. 구분감은 색 차이를 크게 만들기보다 border/shadow 중심으로 잡는다.
  - 니즈 카드 내부 padding은 위/아래 영역 모두 `10px`로 타이트하게 맞췄다. 문의사항 위 구분선은 카드 외곽선과 같은 `#dededb`.

## Operating Rules

- 이사님이 의견을 물으면 바로 구현하지 말고 판단/추천 후 `적용할까요?`.
- 이사님이 `해줘`, `적용해`, `진행해`, `응`처럼 실행 지시하면 바로 구현.
- 작은 CSS/타이포 조정: 테스트는 마지막 묶음에서만.
- DOM/기능 변경: `bun run typecheck` 우선.
- 큰 레이아웃 변경: 필요할 때만 Playwright screenshot 1회.
- 장시간 UI 작업은 1~2시간 단위로 새 세션 전환 권장.
